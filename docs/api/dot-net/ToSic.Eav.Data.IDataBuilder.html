    Interface IDataBuilder

    A data builder which will generate items for a specific type. In many cases it will also take care of auto increasing the id and more.

    This is more efficient than using the bare bones ToSic.Eav.Data.IDataBuilderInternal

    Namespace: ToSic.Eav.Data
    Assembly: ToSic.Eav.Core.dll
    Syntax
    [InternalApi_DoNotUse_MayChangeWithoutNotice("in development for v15")]
public interface IDataBuilder
    Remarks
    • Added in v15 to replace the previous IDataBuilder which is now internal
